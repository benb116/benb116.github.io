Features

Works Silently MusiNotify runs seamlessly in the background, sending notifications without disturbance.
Easy to Customize Preferences allow for complete control over notification content and style, as well as options for the app itself.
Auto-Updates Checks for new versions and alerts the user with update information. Installs new versions by itself.
Smart Handling Fine grain management over notifications in the sidebar. Adds and removes notifications to match user's preferences.
